About

This is a sheet music analyzer designed by a team of Pitzer College students in 2021. It allows you to take a picture or scan of any piece of sheet music you might have and turn it into 3 separate file types:

  1. PDF: This is a clean document of your music that can easily be stored and shared
  2. MIDI: This is a digital music file that can be read by programs such as GarageBand in order to edit notes and virtually play the piece as any instrument
  3. LilyPond: This is a digital markup language that has a small file size and can easily be edited and then exported as a PDF, PNG, or MIDI file using a program that you install or through a free online compiler such as LilyBin
Our web app utilizes two models: an open-source semantic model designed to read an individual line of music and translate that to a semantic vocabulary, and a YOLO model that we use to separate multi-lined scores for the semantic model to read one by one. These separated lines are then parsed back together into our final PDF, MIDI, and LilyPond formats.
